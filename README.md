# igroot 组件库

## 链接

[首页](http://fe.baishancloud.com/components/)

## 支持环境

- React v16.8
- antd v4.0

### 安装

```bash
yarn add bsy-components

// 使用组件
import { SelectTags } from 'bsy-components';
```

### 示例

```jsx harmony
import { SelectTags } from 'bsy-show';

export default () => <SelectTags />;
```
